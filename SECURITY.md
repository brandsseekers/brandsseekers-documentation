# Security Policy

## Reporting a Vulnerability

Brands Seekers takes the security of its platform and public machine-readable resources seriously.

If you believe you have discovered a security vulnerability affecting Brands Seekers, please report it privately.

Email:

security@brandsseekers.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Proof of concept (if available)
- Suggested remediation (optional)

## Responsible Disclosure

Please do not publicly disclose vulnerabilities until they have been investigated and resolved.

We appreciate responsible security research and will acknowledge valid reports where appropriate.

## Scope

This repository contains documentation only.

Security reports relating to the following are welcome:

- Public APIs
- Product feeds
- Structured data
- Documentation inaccuracies with security implications
- Public machine-readable resources
- Integration endpoints

## Out of Scope

The following are generally considered out of scope:

- Typographical errors
- Feature requests
- SEO recommendations
- Publicly documented behavior
- Third-party services outside Brands Seekers' control
