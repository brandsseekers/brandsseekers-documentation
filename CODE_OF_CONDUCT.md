# Code of Conduct

## Our Commitment

Brands Seekers is committed to maintaining a professional, respectful, and collaborative environment for everyone contributing to this documentation.

## Expected Behavior

Participants are expected to:

- Be respectful and constructive.
- Provide accurate technical information.
- Focus discussions on improving documentation.
- Welcome feedback and corrections.
- Respect intellectual property rights.

## Unacceptable Behavior

The following will not be tolerated:

- Harassment or abusive language.
- Personal attacks.
- Spam or promotional content.
- Misrepresentation of Brands Seekers resources.
- Publishing confidential information.

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation contributions
- Community interactions relating to this repository

## Enforcement

Repository maintainers may edit, reject, or remove contributions that do not follow this Code of Conduct.

Repeated violations may result in restricted participation.
