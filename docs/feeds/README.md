# Feed Documentation

Brands Seekers publishes several machine-readable product feeds designed for search engines, AI assistants, commerce platforms, shopping engines, and integration partners.

Each feed serves a specific purpose while representing the same live catalogue of authentic designer fashion available on Brands Seekers.

---

# Available Feeds

| Feed | Format | Purpose |
|-------|--------|---------|
| AI Product Feed | JSON | AI assistants, LLMs, semantic search, machine-readable product understanding. |
| Merchant Product Feed | XML | Merchant Center, shopping platforms, product syndication. |
| Pinterest Product Feed | XML | Pinterest Catalogs and Shopping integrations. |

---

# Documentation Standards

Every feed specification includes:

- Overview
- Purpose
- Public URL
- Format
- Update Frequency
- Schema
- Field Definitions
- Example Payload
- Best Practices
- Version History

---

# Design Principles

All Brands Seekers feeds are designed to:

- Represent the live catalogue of authentic designer fashion.
- Provide consistent structured product data.
- Support machine-readable consumption.
- Minimize ambiguity across platforms.
- Follow stable, versioned specifications.
- Remain backwards compatible whenever possible.
