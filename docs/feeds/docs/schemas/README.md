# Schema Specifications

Brands Seekers publishes structured schemas that describe products, brands, size guides, and other machine-readable resources used throughout the platform.

These specifications define the structure, meaning, and relationships of data published by Brands Seekers.

---

## Available Specifications

| Specification | Description |
|--------------|-------------|
| Product Schema | Defines the structure of product objects. |
| Brand Schema | Defines brand information and metadata. |
| Size Guide Schema | Defines standardized size guide information. |

---

## Documentation Standards

Each schema specification includes:

- Overview
- Purpose
- Object Structure
- Field Definitions
- Data Types
- Validation Rules
- Examples
- Version History
