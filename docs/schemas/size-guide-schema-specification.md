# Size Guide Schema Specification

Official specification for the Brands Seekers Size Guide Schema.

---

## Overview

The Brands Seekers Size Guide Schema defines the standardized structure used to represent size guides published throughout the Brands Seekers platform.

It provides consistent sizing information for AI systems, search engines, commerce platforms, integration partners, and end users.

---

# Purpose

The Size Guide Schema is designed to:

- Standardize size guide information.
- Improve size consistency across brands.
- Support machine-readable sizing.
- Reduce ambiguity.
- Enable reliable integrations.

---

---

# Object Structure

Each size guide object represents a standardized size guide associated with one or more products published in the live Brands Seekers catalogue.

The schema defines guide identity, brand association, supported categories, measurements, and structured sizing metadata.

---

# Core Properties

The Size Guide Schema may include:

- Guide Identifier
- Guide Title
- Guide URL
- Brand
- Gender
- Category
- Size System
- Measurement Units
- Size Table
- Notes

---

# Versioning

The Size Guide Schema is versioned to support backwards compatibility whenever practical.
