# Product Schema Specification

Official specification for the Brands Seekers Product Schema.

---

## Overview

The Brands Seekers Product Schema defines the structure and meaning of product data published throughout the Brands Seekers platform.

It provides a standardized representation of product information for AI systems, search engines, commerce platforms, and integration partners.

---

# Purpose

The Product Schema is designed to:

- Standardize product information.
- Maintain consistent data structures.
- Improve interoperability.
- Support machine-readable product understanding.
- Enable reliable integrations.

# Object Structure

Each product object represents a single product published in the live Brands Seekers catalogue.

The schema defines product identity, pricing, categorization, media, availability, structured attributes, and AI-related metadata.

---

# Core Properties

The Product Schema may include:

- Product Identifier
- Product Title
- Description
- Brand
- Product URL
- Images
- Price
- Currency
- Availability
- Category
- GTIN
- Color
- Gender
- Size
- Product Highlights
- Product Details
- Related Products
- Size Guide
- Style Keywords
- Occasion Keywords
- Search Intent

---

# Versioning

The Product Schema is versioned to support backwards compatibility whenever practical.
