# robots.txt Specification

Official specification for the Brands Seekers robots.txt resource.

---

## Overview

The Brands Seekers robots.txt file defines crawler access policies for publicly accessible content published by Brands Seekers.

It provides guidance to search engines and automated crawlers while supporting efficient discovery and indexing of the live catalogue.

---

# Purpose

The robots.txt resource is designed to:

- Guide compliant web crawlers.
- Improve crawl efficiency.
- Protect non-public resources.
- Advertise sitemap locations.
- Support search engine discovery.
