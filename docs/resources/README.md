# Resource Specifications

Brands Seekers publishes several machine-readable resources that support search engines, AI assistants, web crawlers, and integration partners.

These resources improve discoverability, interoperability, and machine understanding of the Brands Seekers platform.

---

## Available Resources

| Resource | Description |
|----------|-------------|
| robots.txt | Search engine crawling directives. |
| llms.txt | AI assistant and LLM guidance. |
| XML Sitemaps | Discovery of public content. |
| Localized Resources | Language-specific machine-readable resources. |

---

## Documentation Standards

Each resource specification includes:

- Overview
- Purpose
- Public URL
- Format
- Update Frequency
- Structure
- Best Practices
- Version History
