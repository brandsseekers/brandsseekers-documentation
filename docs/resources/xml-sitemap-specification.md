# XML Sitemap Specification

Official specification for the Brands Seekers XML Sitemap resources.

---

## Overview

Brands Seekers publishes XML Sitemaps that help search engines and automated systems discover public pages and resources available throughout the live catalogue of authentic designer fashion.

The sitemaps are generated automatically and reflect the current state of publicly accessible content.

---

# Purpose

The XML Sitemap resources are designed to:

- Improve search engine discovery.
- Advertise publicly available URLs.
- Accelerate indexing.
- Organize website content.
- Support AI and crawler navigation.

# Sitemap Types

Brands Seekers may publish multiple sitemap types, including:

- Product Sitemaps
- Category Sitemaps
- Brand Sitemaps
- Content Sitemaps
- Page Sitemaps
- Post Sitemaps
- Video Sitemaps
- Image Sitemaps (where applicable)

# Update Frequency

Sitemaps are regenerated automatically as the live catalogue and public content evolve.

# Intended Consumers

- Search Engines

# Best Practices

Consumers should:

- Fetch the sitemap index first.
- Respect standard XML Sitemap protocols.
- Avoid assuming sitemap ordering.
- Periodically refresh cached sitemap data.
