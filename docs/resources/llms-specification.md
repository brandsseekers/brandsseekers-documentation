# LLMS Specification

Official specification for the Brands Seekers LLMS resource.

---

## Overview

The Brands Seekers LLMS resource provides a machine-readable index of public resources intended for AI assistants, large language models (LLMs), search engines, and automated knowledge retrieval systems.

It serves as a central discovery document that helps AI systems locate structured resources published by Brands Seekers.

---

# Purpose

The LLMS resource is designed to:

- Help AI systems discover public resources.
- Advertise machine-readable documentation.
- Improve AI interoperability.
- Centralize public resource discovery.
- Support future AI integrations.
