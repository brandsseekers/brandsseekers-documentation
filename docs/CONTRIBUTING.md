# Contributing

Thank you for your interest in improving Brands Seekers Documentation.

This repository contains the official technical documentation for the public machine-readable resources published by Brands Seekers.

---

## Contribution Principles

All contributions should:

- Maintain technical accuracy.
- Reflect production resources.
- Use consistent terminology.
- Follow the Brands Seekers Documentation standards.
- Preserve backwards compatibility whenever practical.

---

## Documentation Requirements

Before submitting changes, ensure that:

- Information reflects the current production platform.
- Public URLs are correct.
- Examples are valid.
- Markdown formatting is consistent.
- Related specifications are updated where necessary.

---

## Scope

This repository documents public resources only.

Proprietary source code, internal infrastructure, implementation details, and private systems are intentionally excluded.

---

## Versioning

Documentation updates should preserve the existing version history and follow the documented versioning policy.
